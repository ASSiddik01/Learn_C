### **Basic Structure**

```bash

#include <stdio.h>

int main()
{
    printf("hello World");
    return 0;
}

```

### **Printf - Print the output**

```bash
printf("Enter number a");
```

### **Scanf - Get the input from user**

```bash
scanf("%d", &a);
```

> Here, %d is the referance of &a and &a to assign the value in "a" variable
